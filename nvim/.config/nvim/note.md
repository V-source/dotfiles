comando mas usados.

`<leader>xx`: Toggle quickfix window.
`<leader>ff`: para buscar archivos del proyecto.
`<leader>fb`: para buscar los buffers que he utilizado durante la sesion.
`<leader>j`: para buscar palabras en un scope limitado del buffer activo.
`;`: para abrir el arbol de carpetas y archivos.
`:`: para cerrar el arbol de carpetas y archivos.
`/`: para buscar palagbras en todo el buffer
`<leader>fg`: para hacer busqueda global de palabras con fzf grep.
`<leader>fi`:. similar a `/`.
`<leader>fw`: busca en todo el proyecto la palabra debajo del cursor o la que escribas.
`<leader>pt`: para abrir el gestor de notas del proyecto.
